USE 
========================

本文将介绍如何通过USE指令在PolarDB-X中切换当前连接的默认数据库。

背景信息
----

PolarDB-X支持访问同一PolarDB-X实例下的多个不同的数据库，就如同单机MySQL的跨数据库查询。 通常，PolarDB-X登录时需要指定一个DB_NAME作为默认数据库。您可以使用USE语句动态切换当前Schema，方便同时管理多个数据库。

语法 
-----------------------

```sql
USE db_name   
```



示例 
-----------------------

您可以使用如下语法将当前的默认库切换到`NEW_DB`。

```sql
USE NEW_DB
```
